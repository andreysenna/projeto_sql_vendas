# Projeto SQL – Análise de Vendas

## 📌 Objetivo

Este projeto tem **um único objetivo**: comprovar meus conhecimentos em SQL para vagas de análise de dados.

Aqui eu demonstro na prática os principais conceitos da linguagem:

- `SELECT`, `WHERE`, `ORDER BY`, `LIMIT`
- `GROUP BY` com funções agregadas (`SUM`, `AVG`, `COUNT`)
- `LIKE` (wildcards)
- `CREATE VIEW`
- `INNER JOIN` e `LEFT JOIN`
- `HAVING`

---

## 🧠 O que foi feito por mim

**Todas as queries SQL foram escritas por mim.**

Eu criei as consultas, testei, ajustei e organizei o notebook para mostrar a evolução do meu aprendizado.

## 🤖 O que foi auxiliado por Inteligência Artificial

Recorri à IA para:

- Criação dos dados fictícios da tabela `vendas`
